# Internfactory Backend Application
The Backend Application is based on the spring boot version 2.6.13
And JAVA version 11
The database used is H2 Database


This Application consist of all the API required for the complete working of the Internfactory Platform where user can host the internships and the applicant user can apply the internships hosted on the platform.
Hosted On - https://internfactory.up.railway.app

To access the APIs
